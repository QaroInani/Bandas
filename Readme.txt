Bandas Musicales version 1.1 Beta
Software de almacenamiento de Bandas Musicales usando bases de datos.

El software presentado almacena información sobre Bandas musicales que se clasifican de acuerdo al estilo musical al que pertenecen (Jazz, Rock, Pop, etc).
Al ejecutar el programa se abre la ventana principal la cual contiene los siguientes elementos:

- Botón 'Nueva Banda':
    Permite al usuario crear una nueva banda musical, para lo cual se abre una ventana secundaria solicitando los datos necesarios; nombre de la banda, estilo musical al que pertence, integrantes, éxitos, discos y si la banda está vigente o no. El campo Nombre de la banda es obligatorio y el campo vigente debe ser respondido con las palabras SI o NO, o nada en el caso de no tener la información. Luego de 'aceptar', se mostrará mensaje de confirmación de la operación y la banda se mostrará inmediatamente en la ventana principal.

- Botón 'Editar Banda':
    Permite al usuario editar la información de una banda, para lo cual previamente debe seleccionarse el elemento o fila a modificar. Para lo cual el programa solicita al usuario seleccionar algun dato, luego se abre una ventana secundaria con la información ya existente de la banda, que se puede modificar con las mismas condiciones de la opción 'Nueva Banda', el campo nombre es obligatorio. Luego de 'aceptar', se muestra mensaje de confirmación de la operación y los cambios se mostrarán inmediatamente en la ventana principal.

- Botón 'Eliminar Banda':
    Permite al usuario poder eliminar una banda, es decir, borrar toda la información existente de la banda (fila completa), para lo cual se solicita haber seleccionado alguna fila previamente, se solicita al usuario confirmar si desea eliminar esa fila, luego de 'aceptar' se muestra mensaje de confirmación de la operación y la banda se elimina inmediatamente de la ventana principal.

- Botón 'Nuevo Estilo':
    Aún no se implementa exitosamente.

- Botón 'Editar Estilo':
    Aún no se implementa exitosamente.

- Botón 'Eliminar Estilo':
    Aún no se implementa exitosamente.

- Buscar:
    Permite al usuario filtrar la información de las bandas de acuerdo a alguna letra o palabra ingresada, el buscador fitra en todas las columnas.

- Buscar por estilo:
    Permite al usuario filtrar la información de acuerdo al estilo musical al que pertenecen las bandas.

- Grilla:
    Se muestra al usuario toda la información contenida en la base de datos correspondiente a las bandas, en la cual se va mostrando inmediatamente cualquier cambio realizado.


NOTA: La aplicación aun se está desarrollando, por lo que este manual podría estar sujeto a cambios.
